# ⭐ HOW TO REVIEW MISTAKES (THE SECRET SAUCE)
For every wrong question:

1. **Label the type**  
   (Inference, boundary, systems, quadratics, etc.)

2. **Why did I miss it?**  
   - Misread?  
   - Rushed?  
   - Didn’t know rule?  
   - Fell for trap?  

3. **Write the correct reasoning**  
   (1–2 sentences)

4. **Do 2 similar questions**  
   (Khan Academy or UWorld)

This is how you improve FAST.
