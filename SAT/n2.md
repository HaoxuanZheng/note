# 🟦 **A. Core Desmos Skills You MUST Know**

### **1. Graphing equations instantly**
Type directly:
```
y = 2x + 5
y = x^2 - 4x + 3
y = |x - 3|
y = 2^(x-1)
```
Desmos auto‑graphs and shows intercepts visually.

**Use for:**  
- Quadratics  
- Systems  
- Absolute value  
- Exponentials  
- Piecewise functions  

---

### **2. Finding intersections (solutions)**
Click the intersection point → Desmos shows the exact coordinates.

**Use for:**  
- Solving systems  
- Solving equations like `2^x = 5x`  
- Word problems with two functions  

---

### **3. Using the Table feature**
Click the gear icon → “Convert to Table”.

**Use for:**  
- Evaluating expressions quickly  
- Checking solutions  
- Seeing patterns in sequences  

---

### **4. Graphing inequalities**
Type:
```
y <= 2x + 3
x > 4
```
Desmos shades the region.

**Use for:**  
- Linear inequalities  
- System of inequalities  
- Feasible region problems  

---

### **5. Checking algebra**
After solving by hand, plug your solution into Desmos:
```
x = 7
```
Then type the original equation to verify.

---

### **6. Graphing both sides of an equation**
For equations like:
```
2^x = 5x
```
Graph:
```
y = 2^x
y = 5x
```
Intersection = solution.

---

### **7. Visualizing absolute value**
Graph:
```
y = |2x - 3|
```
Use for:
- Solving |ax + b| = c  
- Understanding V‑shapes  

---

### **8. Quick quadratic analysis**
Graph:
```
y = ax^2 + bx + c
```
Desmos shows:
- Vertex  
- Roots  
- Minimum/maximum  

---

### **9. Using sliders (practice only)**
Type:
```
y = a(x - h)^2 + k
```
Desmos asks to create sliders for a, h, k.

**Use for:**  
- Understanding transformations  
- Visualizing vertex form  

---

### **10. Checking word problems**
If a problem gives:
```
Revenue = 20x
Cost = 5x + 200
```
Graph both → intersection = break‑even point.

# 🟦 **B. When to Use Desmos (Saves 2–3 Minutes)**

Use Desmos for:
- Ugly quadratics  
- Systems of equations  
- Exponential vs linear intersections  
- Inequalities  
- Word problems with functions  
- Checking answers  

---

# 🟦 **C. When NOT to Use Desmos**

Avoid Desmos when:
- The algebra is simple (linear equations)  
- You can solve in <30 seconds  
- The question asks for **exact** values (like √3/2)  

---

# 🟦 **D. Desmos Speed Tips**

- Use **zoom** to find hidden intersections  
- Use **trace** to follow curves  
- Use **table** to evaluate quickly  
- Use **copy/paste** for repeated expressions
