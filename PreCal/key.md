# 🟦 **UNIT 1 — Polynomial & Rational Functions (One‑Page Summary)**

### **Core Ideas**
- Leading term controls **end behavior**  
- Zeros + multiplicity control **local behavior**  
- Rational functions: **asymptotes, holes, domain**  
- Polynomial division + Remainder Theorem  
- Inequalities solved using **sign charts**

### **Key Skills**
- Identify end behavior from leading term  
- Determine multiplicity (cross vs bounce)  
- Find vertical/horizontal/slant asymptotes  
- Locate holes (cancelled factors)  
- Solve rational inequalities  
- Match equations ↔ graphs

### **Essential Forms**
- Polynomial: \(f(x)=a(x-r_1)^{m_1}(x-r_2)^{m_2}\cdots\)  
- Rational: \(f(x)=\frac{p(x)}{q(x)}\)

### **Quick Rules**
- Even degree + positive leading → up/up  
- Odd degree + positive leading → down/up  
- Cancelled factor → **hole**  
- Denominator zero (not cancelled) → **vertical asymptote**  
- Degrees equal → horizontal asymptote \(y=\frac{a_n}{b_n}\)

### **Common Mistakes**
- Forgetting to exclude cancelled factors from domain  
- Confusing holes with asymptotes  
- Misreading multiplicity  
- Incorrect sign charts

# 🟦 **UNIT 2 — Exponential & Logarithmic Functions (One‑Page Summary)**

### **Core Ideas**
- Exponential growth/decay  
- Logarithms as inverses of exponentials  
- Solving exponential/log equations  
- Semi‑log plots for linearization  
- Modeling real‑world data

### **Key Skills**
- Identify growth factor vs growth rate  
- Convert between \(ab^t\) and \(ae^{kt}\)  
- Apply log rules  
- Solve equations using logs  
- Interpret slope/intercept of semi‑log plots

### **Essential Forms**
- \(f(t)=ab^t\)  
- \(f(t)=ae^{kt}\)  
- \(\log_b x = \frac{\ln x}{\ln b}\)

### **Quick Rules**
- Growth if \(b>1\), decay if \(0<b<1\)  
- \(\ln(ab)=\ln a + \ln b\)  
- \(\ln(a^k)=k\ln a\)  
- Semi‑log plot linearizes exponentials

### **Common Mistakes**
- Using log rules incorrectly  
- Forgetting domain restrictions  
- Confusing growth factor with growth rate  
- Misinterpreting semi‑log slope

# 🟦 **UNIT 3 — Trigonometric & Polar Functions (One‑Page Summary)**

### **Core Ideas**
- Unit circle values  
- Trig functions + transformations  
- Sinusoidal modeling  
- Inverse trig functions  
- Polar coordinates + graphs

### **Key Skills**
- Evaluate trig at special angles  
- Identify amplitude, period, phase shift  
- Build sinusoidal models from data  
- Convert between polar ↔ rectangular  
- Recognize polar graphs (rose, circle, limaçon)

### **Essential Forms**
- \(y=A\sin(B(x-C))+D\)  
- Period \(=\frac{2\pi}{|B|}\)  
- Polar: \(x=r\cos\theta\), \(y=r\sin\theta\)

### **Quick Rules**
- Amplitude = |A|  
- Vertical shift = D  
- Phase shift = C  
- Rose curves: \(r=a\cos(n\theta)\) or \(r=a\sin(n\theta)\)

### **Common Mistakes**
- Mixing degrees and radians  
- Confusing amplitude vs vertical shift  
- Incorrect polar quadrant  
- Misidentifying period

## **Flashcards — Unit 1**

**Q:** End behavior determined by?  
**A:** Leading term \(a_nx^n\).

**Q:** Multiplicity odd vs even?  
**A:** Odd → crosses; even → touches.

**Q:** Hole occurs when?  
**A:** Factor cancels between numerator and denominator.

**Q:** Horizontal asymptote rule?  
**A:** Degrees equal → ratio of leading coefficients.

**A:** \(b=e^k\).

**Q:** \(\log_b(x^k)\) equals?  
**A:** \(k\log_b x\).

**Q:** Domain of \(\ln(x)\)?  
**A:** \(x>0\).

**Q:** Semi‑log plot linearizes what?  
**A:** Exponential functions.

---

## **Flashcards — Unit 3**

**Q:** Period of \(A\sin(Bx)\)?  
**A:** \(\frac{2\pi}{|B|}\).

**Q:** Polar → rectangular?  
**A:** \(x=r\cos\theta\), \(y=r\sin\theta\).

**Q:** \(\sin(\pi/6)\)?  
**A:** \(1/2\).

**Q:** Amplitude formula?  
**A:** \(|A|\).

## **Flashcards — Unit 4**

**A:** \(\sqrt{a^2+b^2}\).

**Q:** Dot product formula?  
**A:** \(u_1v_1+u_2v_2\).

**Q:** Angle between vectors?  
**A:** \(\cos\theta=\frac{\vec{u}\cdot\vec{v}}{|\vec{u}||\vec{v}|}\).
